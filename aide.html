Configuration du clavier français

Pour le confort de tous, nous vous conseillons de modifier un fichier afin d'éviter de devoir changer la langue de votre clavier à chaque session.

  1. Rendez-vous sur votre bureau, là où se trouve RedM.
  2. Faites un clic droit sur l'icône de RedM.
  3. Cliquez sur "Ouvrir l'emplacement du fichier".
  4. Suivez le chemin : RedM Application Data > Citizen > Platform > Data > Control
  5. Téléchargez le fichier de remplacement : ⬇ Télécharger
  6. Remplacez le fichier default.meta existant dans le dossier Control par le nouveau fichier.
★

Résultat : Votre clavier sera désormais configuré en français dans le jeu. Profitez de l'aventure !
