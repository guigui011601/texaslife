À faire pour plus de simplicité

Pour le confort de tous, nous vous conseillons de modifier un fichier afin d'éviter de devoir changer la langue de votre clavier. Voici la procédure à suivre :


Étapes à suivre :

  1. Rendez-vous sur votre bureau, là où se trouve RedM.
  2. Faites un clic droit sur l'icône de RedM.
  3. Cliquez sur "Ouvrir l'emplacement du fichier".
  4. Une fois dans le dossier, suivez ce chemin d'accès :
    RedM Application Data > Citizen > Platform > Data > Control
  5. Téléchargez le fichier nécessaire en cliquant ici : ⬇️
  6. Remplacez le fichier default.meta existant dans le dossier Control par le nouveau fichier.

Résultat : Votre clavier sera maintenant configuré en français dans le jeu.
